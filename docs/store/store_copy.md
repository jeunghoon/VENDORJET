# 스토어 설명/스크린샷 플랜

## 짧은 설명 (공통)
> Wholesale order management with live dashboards, multi-tenant access, and fast catalog updates.

## 긴 설명 요약
1. **Dashboard**: 실시간 지표, 최근 주문, 다국어 지원.
2. **Orders**: 검색/필터, 상태 관리, 편집·삭제, CSV 목업 업로드.
3. **Products**: SKU/가격/재고 플래그, CSV 일괄 등록, 다중 필터.
4. **Customers**: 등급 분류, 연락처 관리, 초대/권한 시뮬레이션.
5. **운영 품질**: 다크모드, 접근성, 오프라인 대비 목업, 향후 Firebase 연동.

## 스크린샷 플로우
1. 대시보드 (라이트/다크)
2. 주문 목록 + 편집 바텀시트
3. 상품 그리드 + CSV 업로드 버튼
4. 고객 목록 + 등급 필터
5. 설정/테넌시/언어 전환 화면

> 실제 빌드 후 각 플랫폼 해상도에 맞게 캡처 예정. 텍스트는 EN/KR 두 버전 제공.
