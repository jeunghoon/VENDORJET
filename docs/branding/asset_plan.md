# VendorJet 아이콘·스플래시 준비 메모

## 아이콘 방향
- **형태**: 원형 안에 `VJ` 모노그램 + 파스텔 하늘 (`#5AA9E6`) 배경, 파스텔 주황 (`#FFA26B`) 포인트.
- **버전**: 
  1. 풀 컬러(디폴트)  
  2. 모노톤(다크모드 대비)  
  3. 단색 스텐실(투명/단색 배경)  
- **해상도**: 1024px 기반으로 iOS/Android/데스크톱 규격 자동 생성 예정 (`flutter_launcher_icons`).

## 스플래시 화면
- **배경**: 파스텔 하늘 그라디언트 (#5AA9E6 → #B8E1FF).  
- **중앙 요소**: 아이콘 모노그램 + "VendorJet" 워드마크.  
- **하단**: 얇은 로딩 인디케이터(선택), 접근성 위해 대비 4.5:1 이상.
